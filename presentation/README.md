# Presentation Files

- `knowledge_distillation_presentation.md`: full slide content draft.
- `speaker_notes.md`: timed speaking notes for 5-7 minutes.

## How to use quickly
1. Open `knowledge_distillation_presentation.md`.
2. Copy each slide section into Google Slides / PowerPoint.
3. Replace `[INSERT IMAGE]` and `[INSERT TABLE]` placeholders with generated artifacts from `outputs/`.
4. Use `speaker_notes.md` as presenter notes.
